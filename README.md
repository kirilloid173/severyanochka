Next.js + SASS

Для стилизации компонентов и модулей, использовался SCSS

В нужных местах приложение логически разделено на компоненты, для передачи данных между компонентами используется стандартный props "проброс"

Предварительные требования

У вас должен быть установлен Node, без него запуск не сработает. Скачать можно с официальной страницы Node -> https://nodejs.org/en Тип установщика выбирайте в зависимости от вашей операционной системы.

Как запустить проект?

Для запуска проекта скачайте репозиторий в любое удобное для вас место, в вашей операционной системе откройте командую строку, перейдите в папку проекта, пропишите команду npm install, затем, после успешной установки всех нужных пакетов пропишите команду npm run dev, для просмотра проекта достаточно лишь открыть в любом удобном для вас браузере страницу localhost:3000

Проект был сделан за 6 дней, но нужно учитывать мою общую занятость другими собеседованиями и работами, поэтому скорость работы на 20-30% "упала" чем если бы я занимался только этим проектом. При оценке скорости работы это нужно учитывать.
